# Difference Between Manager, Head, and Staff

The hierarchy of instruction within the organization is as follows:

| Role    | Responsibility                                                 |
| ------- | -------------------------------------------------------------- |
| Manager | Provides instructions and direction to Department Heads.       |
| Head    | Implements instructions from the Manager and supervises Staff. |
| Staff   | Executes tasks assigned by the Department Head.                |

Example:
The **Plant Manager** provides direction to the **PPC Head**, who then assigns tasks to PPC staff.

# Request Approval Workflow

For requests involving financial transactions (such as purchases, payments, loans, or allowances), approval must be obtained **up to the Vice President level**.

### Request Approval Flow

| Step                   | Responsible Person              |
| ---------------------- | ------------------------------- |
| Preparation of Request | Staff                           |
| Noted By               | Department Head                 |
| Checked By             | Manager                         |
| Reviewed By            | Accounting / Purchasing Officer |
| Approved By            | Vice President                  |

This process ensures proper authorization and accountability for company expenditures.
